# Onmyoji Wiki

Personal fan-made wiki for Onmyoji.

## Features

* Shikigami information
* Skills and upgrades
* Souls
* Evolution materials
* Audio and voice lines

## Live Demo

Visit the website here:

https://onmyoji-wiki.vercel.app/

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
